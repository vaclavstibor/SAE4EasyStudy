# About
This is a repository with sources for the paper *Effects of Quantitative Explanations on Fairness Perception in Group Recommender Systems* submitted to UMAP 2025 Short paper track. Source codes are based on the [EasyStudy framework](https://github.com/pdokoupil/EasyStudy/tree/feature/recsys2023) enhanced by newly implemented plugin that describes overall flow of the user study.

# Reproducibility
Details on how to run EasyStudy and create instance of the user study are described in the [EasyStudy repository](https://github.com/pdokoupil/EasyStudy/blob/feature/recsys2023/README.md).

The particular plugin from which the study should be created is called `grs2024`.
